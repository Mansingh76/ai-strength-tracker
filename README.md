## 🏋️ Key Features

* Workout logging (sets, reps, weights, RPE)
* Nutrition tracking (calories & macros)
* AI predictions (progression, 1RM, plateau detection)
* Fatigue monitoring with deload suggestions
* Progress charts & multi-user support
